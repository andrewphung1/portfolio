# Images Directory

This directory contains all images used in the portfolio website.

## Required Images

### Root Images
- `portrait.jpg` - Your professional portrait photo (square format, recommended: 400x400px or larger)
- `working.jpg` - Photo of you working on a project (recommended: 800x600px or larger)

### Project Images
Place project images in the `projects/` subdirectory:

#### Autonomous Navigation Robot
- `robot-1.jpg` through `robot-4.jpg` (recommended: 1200x900px or larger)

#### Custom Quadcopter Flight Controller
- `quadcopter-1.jpg` through `quadcopter-3.jpg` (recommended: 1200x900px or larger)

#### Smart IoT Environmental Monitor
- `iot-1.jpg` through `iot-4.jpg` (recommended: 1200x900px or larger)

## Image Guidelines

1. **Format**: Use JPG or PNG format
2. **Optimization**: Compress images for web to improve loading times
3. **Aspect Ratios**: 
   - Portrait: 1:1 (square)
   - Working photo: 4:3
   - Project images: 4:3
4. **File Size**: Keep individual images under 500KB when possible
5. **Naming**: Use lowercase with hyphens (e.g., `project-name-1.jpg`)

## Tools for Image Optimization

- [TinyPNG](https://tinypng.com/) - Compress PNG and JPG images
- [ImageOptim](https://imageoptim.com/) - Mac app for image optimization
- [Squoosh](https://squoosh.app/) - Web-based image compressor

## Adding Your Own Images

1. Replace the placeholder image filenames in `index.html` with your actual image filenames
2. Ensure all image paths are correct and match the file structure
3. Test that all images load correctly in your browser
